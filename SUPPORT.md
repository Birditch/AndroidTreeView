# 支持说明 (Support)

AndroidTreeView 是一个基于 .NET 10 + Avalonia 的桌面应用，通过 ADB 展示已连接 Android 设备的信息。
你可以通过以下方式获得支持：

- **缺陷反馈 (Bug reports)**：创建缺陷反馈，并提供复现步骤和环境信息（操作系统、.NET 版本、ADB 版本、
  设备型号 / 地区 / 固件、Android 版本、连接方式），尽量附上截图与日志。
- **功能建议 (Feature ideas)**：创建功能建议，说明产品问题与期望行为。
- **使用问题 (Usage questions)**：创建聚焦的问题 Issue，并说明已经尝试过什么。
- **ADB / 设备连接问题**：先阅读 [docs/adb-requirements.md](docs/adb-requirements.md) 的安装与排错章节
  （未授权 / 离线 / 找不到 ADB 等常见问题）。
- **安全问题 (Security issues)**：使用 [SECURITY.md](SECURITY.md) 中说明的私密报告方式，不要创建公开 Issue。

请尽量让支持请求聚焦于 AndroidTreeView 本身。以下内容通常超出项目范围，可能会被关闭：

- 通用 .NET / Avalonia / C# 开发问题（建议前往对应上游社区）。
- Android SDK / platform-tools 自身的安装与故障（属于 Google 上游工具）。
- 设备厂商固件、驱动、刷机或特定机型的系统问题。
- 与本项目无关的业务逻辑或二次开发定制。
