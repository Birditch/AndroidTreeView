# 行为准则 (Code of Conduct)

AndroidTreeView 希望成为一个友好、务实、彼此尊重的开源协作空间。参与本项目（Issue、Pull Request、讨论等）时，
请遵守以下原则：

- 保持友善和专业，尤其是在意见不同的时候。
- 将批评聚焦于想法、设计和代码，而不是针对个人。
- 尊重不同的经验水平、语言、文化和工作方式。
- 让讨论围绕项目本身和社区协作展开。
- 不发布骚扰、人身攻击、垃圾信息或歧视性内容。

## 维护者职责 (Maintainer Responsibilities)

维护者可以编辑、隐藏或删除违反这些原则的内容，并在必要时临时或永久限制相关参与行为，以保护项目社区。

## 举报方式 (Reporting)

如果你遇到或发现不当行为，请通过仓库所有者（见 [CODEOWNERS](.github/CODEOWNERS)）可用的 GitHub 联系渠道
私下联系维护者。涉及安全漏洞的问题请改用 [SECURITY.md](SECURITY.md) 中说明的私密报告流程。
